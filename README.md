## 1. 프로젝트 소개
- **서비스명**: 냉장고를 부탁해 : 자취생편
- **소개**: 냉장고 안에 남아 있는 식재료들을 마우스 클릭 한 번으로 바구니에 담아 선택하면, 해당 재료들로 당장 만들어 먹을 수 있는 최적의 레시피들을 추천해 주는 아기자기한 SPA(Single Page Application) 웹 서비스입니다. 
- **핵심 목표**: 1인 가구가 식재료의 보관 기한을 놓치지 않고 낭비 없이 요리할 수 있도록 도와 식비를 절약하고 환경을 지키는 것입니다.

## 3. 실행 방법 (How to Run)
본 프로젝트는 현대적인 ES 모듈(`type="module"`) 스펙으로 작성되었습니다. 브라우저 보안 규정(CORS)으로 인해 로컬 파일(`file:///`) 더블클릭 방식으로 실행할 수 없으며, **반드시 로컬 웹 서버(HTTP)**를 통해 접속해야 정상적으로 작동합니다.

### 방법 A. VS Code Live Server 확장 활용 (권장)
1. VS Code에서 `2026_bigdata_camp` 폴더를 엽니다.
2. 에디터 우측 하단에 위치한 **`Go Live`** 버튼을 클릭합니다.
3. 자동으로 실행되는 주소(`http://127.0.0.1:5500`)를 통해 브라우저에서 사용합니다.

### 방법 B. Node.js (npx) 서버 활용
1. 터미널(Command Prompt 또는 PowerShell)을 실행합니다.
2. 프로젝트 루트 디렉토리(`C:\Users\gram14\Desktop\2026_bigdata_camp`)로 이동합니다.
3. 아래 명령어를 실행합니다:
   ```bash
   npx serve
   ```
4. 실행 후 출력되는 로컬 호스트 주소(예: `http://localhost:3000`)로 브라우저를 열어 접속합니다.
