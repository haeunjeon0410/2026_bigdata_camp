# 냉장고를 부탁해 : 자취생편

> 냉장고 속 재료를 선택하면 지금 만들 수 있는 요리와 부족한 장보기 품목을 추천해 주는 1인 가구 맞춤형 레시피 서비스입니다.

## 배포 링크

**[서비스 바로가기](https://2026-bigdata-camp.vercel.app/)**

## 프로젝트 소개

`냉장고를 부탁해 : 자취생편`은 냉장고에 남은 식재료를 효율적으로 활용할 수 있도록 돕는 웹 서비스입니다. 사용자는 냉장고에서 보유 재료를 선택하고, 보유 재료와 레시피의 매칭률을 기준으로 맞춤형 요리를 추천받을 수 있습니다.

요리에 익숙하지 않은 자취생도 쉽고 재미있게 사용할 수 있도록 냉장고를 직접 여는 듯한 인터랙션과 아기자기한 UI를 적용했습니다.

## 주요 기능

- 냉장고 UI에서 보유 식재료 선택
- 식재료 검색 및 카테고리별 필터링
- 보유 재료 기반 레시피 추천 및 매칭률 확인
- 부족한 재료와 장보기 체크리스트 제공
- 편의점 재료로 만들 수 있는 대체 레시피 추천
- 레시피 상세 정보, 조리 순서, 조리 시간 및 난이도 안내
- 레시피 즐겨찾기와 요리 도감
- 영수증 이미지 분석을 통한 식재료 자동 등록

## AI 기능: 영수증 분석

영수증 이미지를 업로드하면 OpenAI Vision API가 영수증에 표시된 식품 상품을 분석하고, 서비스에서 사용할 수 있는 재료 정보로 변환합니다.

### 처리 흐름

```text
영수증 이미지 업로드
        ↓
Supabase Edge Function 호출
        ↓
OpenAI Vision API 분석
        ↓
상품명·일반 재료명·수량 JSON 반환
        ↓
서비스 재료 데이터와 매칭
        ↓
사용자 확인 후 냉장고에 등록
```

AI가 냉장고 데이터를 바로 변경하지 않고, 분석 결과를 먼저 보여준 뒤 사용자가 선택한 항목만 등록하도록 설계했습니다. 이를 통해 잘못 인식된 상품이 자동으로 저장되는 것을 방지하고 사용자 통제권을 유지합니다.

### 응답 형식

```json
{
  "items": [
    {
      "rawName": "영수증에 표시된 상품명",
      "normalizedName": "서비스에서 사용할 일반 재료명",
      "quantity": "수량 또는 중량"
    }
  ]
}
```

영수증에 실제로 보이는 상품만 분석하도록 제한하며, 비식품 항목·할인·배송비·결제 금액 등은 결과에서 제외합니다. 분석 결과는 중복 제거 및 기존 재료 데이터와의 매칭을 거쳐 냉장고 등록 화면에 표시됩니다.


## 기술 스택

| 구분 | 기술 |
| --- | --- |
| Frontend | HTML, CSS, JavaScript ES Modules |
| UI | 반응형 SPA, CSS Animation, Google Fonts |
| Backend | Supabase |
| AI | OpenAI Vision API (`gpt-4o-mini`) |
| Serverless | Supabase Edge Functions (Deno) |
| Deployment | Vercel |
| Data | JavaScript 기반 레시피·재료 데이터, LocalStorage |

## 프로젝트 구조

```text
2026_bigdata_camp/
├── index.html                         # SPA 진입점
├── style.css                          # 전체 UI 및 애니메이션
├── js/
│   ├── app.js                         # 전역 상태, 라우팅, 화면 렌더링
│   ├── data.js                        # 재료·레시피 데이터 및 매칭 로직
│   ├── ingredient.js                  # 냉장고 재료 선택·검색·필터
│   ├── recommend.js                   # 레시피 추천 및 캐러셀
│   ├── recipe.js                      # 레시피 상세·조리·평가
│   ├── favorite.js                    # 즐겨찾기
│   ├── shopping.js                    # 장보기 체크리스트
│   ├── receipt.js                     # 영수증 업로드·분석·등록
│   └── supabase.js                    # Supabase 및 Edge Function 연동
├── supabase/functions/analyze-receipt/
│   └── index.ts                       # OpenAI Vision 기반 영수증 분석 함수
└── docs/
    ├── PRD.md                         # 제품 요구사항 문서
    └── spec.md                        # 개발 명세서
```

## 사용자 플로우

```text
홈
 └─ 내 냉장고
     ├─ 재료 선택 또는 영수증으로 재료 등록
     └─ 레시피 추천받기
         └─ 레시피 상세
             ├─ 조리 가이드
             ├─ 부족한 재료 장보기
             └─ 즐겨찾기·요리 도감 기록
```

## 기대 효과

- 냉장고 속 재료를 활용해 식비를 절약할 수 있습니다.
- 방치되기 쉬운 식재료의 소비를 유도해 음식물 쓰레기를 줄일 수 있습니다.
- 요리에 익숙하지 않은 사용자도 쉬운 레시피를 빠르게 선택할 수 있습니다.
- 영수증 분석으로 재료를 하나씩 직접 입력해야 하는 불편을 줄일 수 있습니다.

## 한계 및 향후 확장

현재 영수증 분석은 이미지에서 확인되는 상품을 구조화하고, 사용자가 확인한 뒤 냉장고에 등록하는 기능에 초점을 둡니다. 향후 다음 기능으로 확장할 수 있습니다.

- 분석된 재료 기반 자동 분류 및 보관 위치 추천
- 재료 보유량과 유통기한을 고려한 레시피 추천
- 내부 레시피·보관법 문서를 활용한 RAG 기반 추천 근거 제공
- 여러 단계의 도구 호출, 실패 재시도, 사용자 확인을 포함한 에이전트 workflow
