# Product Requirements Document (PRD)

## 1. 프로젝트 소개
- **서비스명**: 냉장고를 부탁해 : 자취생편
- **한 줄 소개**: 냉장고에 남은 재료를 선택하고, 맞춤형 레시피와 부족한 장보기 품목을 추천해 주는 아기자기한 감성의 1인 가구 최적화 자취생 요리 가이드 서비스.

## 2. 프로젝트 목표
- **식재료 효율 극대화**: 남은 재료들을 버리지 않고 활용할 수 있는 최적의 레시피를 제안하여 식비 절약에 기여합니다.
- **음식물 쓰레기 감소**: 냉장고 안에서 방치되다가 상해 나가는 재료를 최소화하여 환경적 요소를 고려한 요리 습관을 형성합니다.
- **요리 진입 장벽 완화**: 요리에 서툰 자취생들을 위해 난이도가 낮고 간편하며, 구하기 쉬운 재료 중심의 요리를 제안합니다.

## 3. 대상 사용자
- 냉장고에 남은 재료를 어떻게 요리해야 할지 막막한 **자취생 및 1인 가구**
- 식비를 절약하고 요리 솜씨를 기르고 싶은 **대학생 및 사회 초년생**
- 복잡한 요리법 대신 쉽고 귀여운 테마의 간편 레시피를 선호하는 **사용자**

## 4. 핵심 기능 (MVP)
### 4.1. 재료 관리
- **보유 재료 선택**: 냉장고 내부 선반과 야채칸을 모방한 귀여운 3D 캐비닛 레이아웃에서 식재료 스티커를 톡톡 클릭하여 선택 상태로 지정(바구니에 담기)합니다.
- **카테고리 분류**: 유제품/달걀, 식사/고기, 신선 야채 등으로 재료 분류 탭(문 주납공간)을 분할 제공합니다.
- **검색 및 필터링**: 냉장고 문 앞 자석 포스트잇 형태의 검색 바에서 실시간 텍스트 매칭으로 원하는 재료를 빠르게 선별합니다.

### 4.2. 레시피 추천
- **보유 재료 기반 정렬**: 사용자가 선택한 재료와 매칭률이 높은 레시피를 우선적으로 추천 순서에 반영합니다.
- **부족한 재료 시각화**: 추천 리스트 카드 상에서 요리에 필요한 재료 중 현재 사용자에게 부족한 재료의 목록과 개수를 명확히 보여줍니다.
- **편의점 꿀조합 레시피 토글**: "마음에 드는게 없어요 😢" 버튼 클릭 시, 부족한 냉장고 재료를 대체할 수 있는 대학가 인기 편의점 모디슈머 6가지 레시피 조합 목록으로 간편하게 스위칭합니다.

### 4.3. 레시피 상세 정보
- **책자형 상세 UI**: 아기자기한 손글씨체 노트 테마로 구성된 화면에서 조리 시간, 난이도, 필요 재료 및 대체 팁을 안내합니다.
- **순차적 가이드**: 요리 시작 시 1번부터 차근차근 따라할 수 있는 레시피 단계를 제공하며, 조리 애니메이션 프로그레스 바를 통해 시각적 재미를 더합니다.

### 4.4. 즐겨찾기 (하트 찜)
- **보관함 저장**: 마음에 드는 레시피 카드나 상세 화면의 하트(❤️) 버튼을 눌러 보관할 수 있습니다.
- **도감 조회**: 요리 도감(📖) 메뉴를 통해 즐겨찾기에 등록된 나만의 레시피 컬렉션을 언제든 모아볼 수 있습니다.

## 5. 추가 기능 (선택 구현)
- **장보기 영수증 생성**: 편의점 조합 등의 레시피 상세 페이지에서 가상 영수증 레이아웃 형태로 부족한 상품 리스트와 예상 합계 금액을 보여주어 바코드로 실시간 모디슈머 요리를 준비할 수 있게 돕습니다.
- **AI 추천 이유 시각화**: 해당 레시피가 현재 선택한 재료 상황에서 왜 적합한지에 대한 맞춤형 AI 진단 한 줄(예: "포실포실한 감자와 양파의 조화가 훌륭해요")을 곁들여 의사결정을 지원합니다.
- **도감 수집 진행률**: 전체 레시피 대비 실제 사용자가 완성해서 시식 평가를 남긴 요리의 발견 비율(도감 완성률 %) 및 조리 횟수를 카운트하여 게임 퀘스트 같은 성취감을 부여합니다.

## 6. 사용자 플로우
```
🏠 Home (시작) 
   └─ 🧺 내 냉장고 (문 열기 -> 검색/카테고리 탐색 -> 재료 스티커 바구니에 담기)
         └─ 🍳 추천 레시피 (매칭율순 추천 카드 -> 마음에 안 들 시 편의점 꿀조합 스위칭)
               └─ 📖 레시피 상세 (필요 재료 메모 -> 조리 시작 -> 요리 완료 및 별점 평가)
                     └─ 📖 요리 도감 (완성한 요리 횟수 체크 & ❤️ 하트 찜 보관함 조회)
```

## 7. 페이지 구성
- **Home (시작 페이지)**: 서비스 웰컴 화면 및 냉장고 문 열기 동선 안내
- **Fridge (내 냉장고)**: 보유 재료 선택, 추가, 삭제, 검색 인터페이스
- **Recipes (추천 목록 페이지)**: 추천 카드 캐러셀 스와이프 기능, 편의점 레시피 토글 기능
- **Recipe Detail (상세 페이지)**: 필요 재료 및 팁, 조리 가이드 순서, 장보기 영수증 표시
- **Cooking & Completion (조리 & 완료 페이지)**: 조리 중 로딩 애니메이션 및 완성 축하 콘페티 효과
- **Rating (별점 평가 페이지)**: 냠냠 시식 피드백 및 요리 도감 수집 연동
- **Book (요리 도감 페이지)**: 총 발견 진행률, 조리 카운트, 하트 즐겨찾기 리스트

## 8. 데이터 구조
### 8.1. 식재료 (Ingredients)
- `id`: 고유 식별자 (예: `'egg'`)
- `name`: 표시 재료명 (예: `'계란'`)
- `emoji`: 그래픽 이미지 아이콘 (예: `'🥚'`)
- `category`: 소속 카테고리 탭 (예: `'dairy'`)

### 8.2. 레시피 (Recipes & Alternative Recipes)
- `id`: 레시피 고유 ID (예: `'r1'`, `'alt1'`)
- `name`: 요리명 (예: `'감자전'`)
- `emoji`: 요리 그래픽 아이콘 (예: `'🥞'`)
- `difficulty`: 난이도 정보 (예: `'쉬움'`, `'보통'`)
- `time`: 조리 예상 시간 (예: `'20분'`)
- `need`: 필수 부합 식재료 ID 배열
- `missing`: 추가 준비해야 할 누락 재료 배열
- `aiReason`: 맞춤 추천 상세 코멘트
- `ingredients`: 정량 상세 리스트
- `steps`: 순차적 조리 스텝 목록
- `cookingSequence`: 쿠킹 애니메이션 파티클 이모지 배열
- `priceList` (편의점 대체제 적용 시): 개별 품목 단가
- `totalPrice` (편의점 대체제 적용 시): 장보기 총합 예산

## 9. 비기능 요구사항
- **SPA 웹 어플리케이션**: 페이지 깜빡임 없는 매끄러운 뷰 전환(SPA 라우팅)
- **반응형 웹 지원**: 모바일 스마트폰 세로 뷰에서부터 데스크톱 크기까지 완벽히 대응하는 그리드와 레이아웃 배치
- **부드러운 애니메이션**: 냉장고 문 스윙, 스티커 선택 튀어오름(Bounce), 요리 완료 흩날리는 종이가루(Confetti) 효과를 통한 사용자 인터랙션 극대화

## 10. MVP 범위
- 초기 MVP 릴리즈는 정적 JSON 기반의 데이터 조회를 구현 모델로 채택합니다.
- 데이터 모델과 로직 인터페이스를 명확하게 캡슐화하고, 차기 버전에서 Supabase 클라우드 데이터베이스 API 호출 모듈로 투명하게 전환 가능하게 레이어를 분할 배치합니다.
